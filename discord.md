# Discord AI Bot

An AI-powered Discord bot built using Python and Google's Gemini API.

## Features

- Responds when mentioned
- AI-powered conversations
- Easy to extend

## Technologies

- Python
- discord.py
- Google Gemini API
- python-dotenv

## Installation

Clone the repository:

```bash
git clone https://github.com/HackerBoyHub/Discord-Voice-bot.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```text
DISCORD_TOKEN=your_token
GEMINI_API_KEY=your_api_key
```

Run the bot:

```bash
python bot.py
```
